# C3的设计目标
---

## 设计目标:
* 过程式语言, 秉持务实精神, 以完成任务为首要目标
* 极简主义, 不引入任何不必要或冗余的特性
* C味足
* C程序员们的下一门语言
* 与C无缝集成
* 人体工程学的设计
* 惰性的数据
* 0为初始化(ZII)
* 脚踏实地

*“Zero Is Initialization” is an idiom where types and code are written so that the zero value is a meaningful, initialized state.*

## 特性:
* 完全C接口
* 模块系统
* 操作符重载
* 泛型
* 契约式设计
* 0开销错误处理
* 语意宏
* 一等公民的 SIMD 向量类型
* 结构体子类型化
* 安全的切片
* 安全的遍历(foreach)
* 好使的行内汇编
* 跨平台
* LLVM后端

# C3的背景:
C3是C的进化, 一个极简的系统编程语言, 沿用C范式

C3最初是 Bas van den Berg 对C2语言的一个实验性分支, 后来在语法, 错误处理, 宏, 泛型以及字符串等方面都经历了显著演进。
